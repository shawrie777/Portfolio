QMaths

I started this project with the intention of creating a maths library that could handle all the necessary linear algebra for computer graphics.

Most maths libraries tend to use matrices to handle rotations but I decided it would work better with quaternions instead.

This library contains everything needed to handle translation, rotation and scaling, and allows vectors and matrices of any size.

The repo can be found here.

QGraphics

I built this library to provide a quick way to build graphical programs using openGL in windows. It is built on QMaths and has features to create a wide variety of built-in 3D shapes, lighting and materials, and even allows importing FBX models created in Blender.

This covers everything needed for a simple 3D graphics program, including mouse and keyboard input, camera controls and the ability to precisely move, rotate or scale objects.

The repo can be found here.

Draughts

This is my version of the classic boardgame, built using my own graphics library, QGraphics.

It features 3D movement, with each piece visibly lifting off the board, and a shadow effect underneath. It also uses multiple materials for the pieces to indicate kings and selected pieces.

The repo can be found here.

Algebra System

Inspired by Wolfram Alpha, I'm building this with the intention of it being able to solve as many equations as possible.

It uses regular expressions to understand the users input and creates a set of nested expressions to prioritise its order of operations. It then works from the deepest nested layer outwards to find a solution. This is still a work in progress.

The repo can be found here.
